# Van Gogh Website

This project is a personal website showcasing the life and works of Vincent van Gogh. It includes a biography, a gallery of his paintings, and a contact form.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Technology Used](#technology-used)

## Installation

To get started with this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/yourusername/vangogh-website.git
cd vangogh-website
npm install
```

## Usage

To start the development server, run:

```bash
npm start
```

This will launch the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Features

- **Biography**: A detailed biography of Vincent van Gogh, highlighting key moments in his life and career.

- **Gallery**: A collection of his most famous paintings, with high-quality images and descriptions.

- **Contact Form**: A form to get in touch with the website owner, allowing visitors to send messages directly from the site.

- **Responsive Design**: The website is fully responsive and works well on all devices, including desktops, tablets, and mobile phones.

- **Interactive Elements**: Includes interactive elements such as image sliders and hover effects to enhance user experience.

## Technology Used

- **HTML5**: For structuring the content of the website.

- **CSS3**: For styling the website and making it visually appealing.

- **JavaScript**: For adding interactivity and dynamic content.

- **React**: A JavaScript library for building user interfaces.
